About PowSyBl

Powsybl (Power System Blocks) is an open source framework written in Java, dedicated to electrical grid modelling. It provides everything necessary to the network representation, but also its simulation. The power system blocks may be used through scripts for a quick shot, but also be assembled to build state of the art applications.

Powsybl is part of the LF Energy Foundation, a project of The Linux Foundation that supports open source innovation projects within the energy and electricity sectors.

One major aim of the project is to make it easy to write complex software for power system simulation and analysis. For example, using Powsybl one can create applications able to:

  • handle a variety of formats, such as CGMES for European data exchanges
  • perform power flow simulations and security analyses on the network
  • soon to come: perform dynamic simulations on the network

Another key characteristic of Powsybl is its modular approach, that enables developers to extend or customize its features. Providing such a design was at the core of the opensource approach: anyone can customize Powsybl's features for his own use.
